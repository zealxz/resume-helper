# 🚀 Resume-Helper: 拒绝“AI 塑料味”的诚信简历定制引擎

> **“不要让 AI 的幻觉，毁掉你真实的职业生涯。”**

Resume-Helper 是一款专为真实求职者打造的本地 AI 简历优化技能（Skill）。不同于市面上盲目润色、胡编乱造的 AI 工具，我们采用极其严格的**「反幻觉协议」**和**「文本洁癖标准」**。只挖掘你真实的过往经历，精准对齐目标岗位 JD，并一键生成 HR 友好的精美 PDF。

---

## ⚡ 快速开始 (Quick Start)

只需三行命令，立刻在本地启动你的私人求职优化引擎：

```bash
# 1. 安装核心技能与 PDF 导出依赖
clawhub install resume-helper
clawhub install md-to-pdf-cjk

# 2. 根据目标岗位 JD 关键词，安全重组你的简历
node optimize-resume.js my-resume.md "Python, 微服务, 高并发, 敏捷开发" "目标公司名称"

# 3. 将优化后的纯净版 Markdown 一键转为原生排版的 PDF
node export-pdf.js my-resume-optimized.md

🔥 核心特性 (Features)

    🛡️ 独创「反幻觉协议」：绝对禁止捏造工作经历和项目数据！引擎只被允许提取、转化和重组你现有的经验，将其翻译成目标岗位最想看到的“行话”，保护你的职业信誉。

    📊 ATS 匹配度体检报告：拒绝盲盒优化！每次运行都会单独生成一份战报，直观展示修改前后的 ATS（简历自动筛选系统）匹配度提升（例如：Match Score 20% ➡️ 85%）。

    🧹 「文本洁癖」输出：全面剔除大模型爱用的星号（*）、无意义的强调和 AI 话术。输出极其干净、专业的商务文本结构，绝无“AI 塑料味”。

    📄 原生中文 PDF 导出：完美内置 CJK 字体支持，抛弃沉重的无头浏览器或 LaTeX，极速将优化结果生成 HR 最爱的标准 PDF 格式及专属求职信（Cover Letter）。

🎯 典型使用场景

    🎓 应届生求职：不知道怎么写简历？提取你在校期间的毕设和实验，将其精准转化为大厂 JD 里的“项目落地能力”和“架构拆解思维”。

    🔄 转行与跳槽：打破行业壁垒，用目标行业的关键词重新包装你的底层通用能力。

    🚀 极致海投策略：面对 10 家不同侧重点的公司，利用本地脚本在 5 分钟内生成 10 份完美对齐的定制化 PDF 简历。

💻 极客技术亮点

    极致轻量：底层基于 Python reportlab 引擎与原生 CJK 字体构建 PDF，无需安装庞大的 Puppeteer 或复杂的环境。

    极低门槛：只需 2GB RAM 即可在任何本地终端流畅运行。
